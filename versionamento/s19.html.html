1.Protocolos de comunicação em rede

Protocolos de comunicação em rede são um conjunto de regras e convenções que determinam como os dados são transmitidos e recebidos em uma rede. Eles garantem que a comunicação entre dispositivos seja eficiente e confiável.

2.Http

HTTP, que significa Protocolo de Transferência de Hipertexto, é a base da comunicação de dados na web. É o que os navegadores usam para solicitar e transmitir páginas web e outros recursos dos servidores

3.Ftp

FTP, que significa Protocolo de Transferência de Arquivos (File Transfer Protocol em inglês), é um protocolo utilizado para transferir arquivos entre um cliente e um servidor em uma rede. Ele permite que você envie e receba arquivos de e para servidores, o que é útil para carregar arquivos em um site, fazer backups ou compartilhar grandes quantidades de dados.

4.SMTP

SMTP, que significa Protocolo Simples de Transferência de Correio (Simple Mail Transfer Protocol em inglês), é um protocolo padrão para o envio de e-mails pela internet. Ele é utilizado para enviar mensagens de e-mail de um cliente de e-mail (como Outlook, Gmail, Thunderbird) para um servidor de e-mail, e também para transferir e-mails entre servidores.

5.Modelo TCP/IP

O modelo TCP/IP (Transmission Control Protocol/Internet Protocol) é um conjunto de protocolos de comunicação que fundamenta a arquitetura da internet e das redes de computadores. Ele é chamado de "modelo" porque define um conjunto de regras e procedimentos para a comunicação de dados entre dispositivos em uma rede.
